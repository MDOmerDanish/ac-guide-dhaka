# CLAUDE.md

This file provides guidance to Claude Code (claude.ai/code) when working with code in this repository.

## Repository Purpose

This is a content repository containing markdown guides and social media post drafts for buying air conditioners in Dhaka, Bangladesh. There is no build system, test suite, or code to run.

## Content Overview

- `general_ac_buying_guide_dhaka.md` — Comprehensive buying guide focused on General (O-General / Fujitsu General) ACs, with alternatives (Gree, Haier, Walton). Covers counterfeiting warnings, official model/pricing matrix, capacity sizing for Dhaka, and purchase checklist.
- `buet_alumni_post.md` — Draft social media post for the BUET alumni community seeking peer advice on the same purchase decision.

## Key Content Facts to Preserve

- **Esquire Electronics Limited** is the sole authorized General distributor in Bangladesh.
- General's wall-mounted inverter models (ASGG series) are cooling-only; Hot & Cool wall-mount units (ASG series) are non-inverter. No official wall-mounted General inverter Hot & Cool unit exists for the Bangladesh market.
- The user's stated priority: **reliability and after-sales support over price** (budget is not a constraint).
- Prices are in BDT and are estimates; they fluctuate with seasonal promotions.
